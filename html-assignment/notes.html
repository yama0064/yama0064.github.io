Tags

Headings
We can use the h1 tag for the most important heading of the page. For the next level of heading we have h2, h3, h4, and h5. The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Paragraphs
For adding a paragraph of text we can use the p tag. The browser will add a space before and after the content of the paragraph.
Quote
A small quotation embedded within other content. "You become what you think."
Emphasis
A string of emphasized, slightly more important text. Screen readers will change their voice for this text.
